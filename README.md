# Asteroids

A classic **Asteroids** arcade game built with **Python** and **Pygame**.

## Features

- 🚀 Player-controlled spaceship
- ☄️ Asteroid spawning and splitting
- 🔫 Bullet shooting mechanics
- 💥 Collision detection

## Requirements

- Python 3.x
- Pygame

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/priteesh-rahate/astroid-game.git
   cd astroid-game
   ```

2. Install the dependencies:

   ```bash
   pip install pygame
   ```

## Usage

Run the game with:

```bash
python main.py
```

## Controls

| Key | Action |
|-----|--------|
| **W** | Thrust forward |
| **A** | Rotate left |
| **D** | Rotate right |
| **Space** | Shoot |
